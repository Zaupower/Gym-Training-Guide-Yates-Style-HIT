# Changes from v2:
#   - TITLE: added to each day block → pre-fills session title
#   - NOTES: added to each day block → pre-fills session notes
#   - Cardio exercise renamed to "AeroBike Sprints" (specific machine)
#   - Warm-up vs sprint sets clearly labelled in comments
#
# CARDIO FORMAT:
#   AeroBike Sprints | cardio | w:1x90sec, 3x20sec
#     - w:1x90sec = 1 warmup set  — 90sec easy spin (shows as "Warm" in app)
#     - 3x20sec   = 3 working sets — 20sec all-out sprint (shows as "Work" in app)
#   Total ~6 min. Log peak watts in the session notes after training.
#
#   Other cardio machines use the same format — just change the name:
#     Stationary Bike  | cardio | w:1x5min, 1x20min
#     Treadmill Run    | cardio | w:1x5min, 1x25min
#     Jump Rope        | cardio | w:1x90sec, 5x1min
#     Rowing Machine   | cardio | w:1x3min, 3x4min
#
# UPLOAD ORDER — activate each plan when its date arrives:
#   Plan 01  Foundation       4w   2026-05-04
#   Plan 02  Cycle 1 Push     6w   2026-06-01
#   Plan 03  Cycle 1 Deload   2w   2026-07-13
#   Plan 04  Cycle 2 Push     6w   2026-07-27
#   Plan 05  Cycle 2 Deload   2w   2026-09-07

#   --- Week off: 2026-11-02 to 2026-11-08 (no plan, no cardio) ---
#   Plan 07  Cycle 4 Push     6w   2026-11-09
#   Plan 08  Cycle 4 Deload   2w   2026-12-21
#   Plan 09  Cycle 5 Push     6w   2027-01-04

#   Plan 12  Cycle 6 Deload   2w   2027-04-12
#   --- Week off: 2027-04-26 to 2027-05-02 (no plan, no cardio) ---
#
# WEEKLY STRUCTURE:
#   Mon  Chest + Back
#   Tue  Cardio (push cycles only — dropped in deload)
#   Wed  Legs
#   Thu  Rest
#   Fri  Shoulders + Arms + Calves
